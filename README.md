# Desktop Environment

A modern, interactive web-based desktop environment built with React, TypeScript, and modern web technologies. Features a fully functional terminal, AI chat application, and PDF viewer.

## Features

- **Interactive Terminal**: Command-line interface with file operations, system commands, and utilities
- **AI Chat**: Local LLM integration with WebGPU acceleration for offline AI conversations
- **PDF Viewer**: Built-in document viewer for seamless file reading
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern UI**: Glass-morphism design with smooth animations

## Applications

### Terminal
- File system navigation and operations
- System information commands
- Fun commands (matrix effect, quotes, jokes)
- Window management controls
- Command history and autocomplete

### AI Chat
- Local language model running in browser
- WebGPU acceleration when available
- Document analysis capabilities
- Offline functionality

### PDF Viewer
- View and interact with PDF documents
- Zoom, navigation, and search functionality
- Integrated with terminal commands

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Start development server: `pnpm dev`
4. Open in browser and start exploring!

## Commands

Type `help` in the terminal to see all available commands, or try:
- `neofetch` - Display system information
- `ls` - List files and applications
- `matrix` - Enter the Matrix!
- `ai` - Open AI Chat application
- `pdf` - Open PDF Viewer
